# EDGE 2026 Google + Meta Execution Playbook (University Tie-ups)

## 1) Objective
- Drive `EDGE (BSc)` admissions for partner universities with a scalable, quality-first performance system.
- Build a repeatable structure that works across `21+ universities`, multiple states, and mixed admission timelines.
- Optimize for funnel progression, not just leads:
  - `Lead -> Appointment -> Demo -> Enrollment`

## 2) Core Strategy (Non-Negotiable)
- `Google = demand capture` (high intent, search-led, location + program-specific)
- `Meta = demand creation + nurture` (awareness, consideration, remarketing, appointment/demo movement)
- Do not mirror Google and Meta monthly trends.
- Creative and landing pages must be `university-specific` or `cluster-specific` to protect lead quality.
- Track and optimize by `program + university + source + stage`, not only CPL.

## 3) EDGE Segmentation Framework (How to Operate 21+ Universities)
Use a tiered rollout. Do not launch all campuses with the same intensity on Day 1.

### Tier A (Primary budget focus)
Criteria:
- `Admission 2026 Open`
- Start date confirmed / near-term
- Clear payment flow and fee structure available
- Brochure/images/video assets available
- Higher intake (`300+` preferred) or strong brand pull

Examples from your sheet (likely Tier A candidates, validate final):
- `Medhavi Skill University` (intake ~600)
- `Saraswati Group of Colleges` (intake ~510)
- `Lovely Professional University` (intake ~480)
- `Arka Jain University` (intake ~420)
- `AMET University` / `MediCaps` / `Universal SkillTech` / `Sanjivani` (based on readiness + intake)

### Tier B (Controlled launch / test budget)
Criteria:
- Open admissions but missing some assets, pending exact start dates, or unclear process/refund details
- Mid-lower intake (`100-300`)

### Tier C (Hold / remarketing only until ready)
Criteria:
- Admission status not open / incomplete
- Missing fees, brochure, application flow, or owner/campus confirmation

## 6) Google Ads Strategy (EDGE)

### 6.1 Campaign Stack (Recommended)
Keep this structure consistent across states/regions.

1. `Search | Brand | Emversity + EDGE`
- Keywords: emversity edge, emversity bsc, emversity allied health

2. `Search | University Brand | [University Name] + Program`
- Example: `AMET allied health admission`, `MediCaps AHP admission`, `LPU paramedical courses`
- Separate campus names if needed (e.g., Chennai vs Bengaluru)

3. `Search | Generic High Intent | Program + Admission`
- `bsc allied health admissions`
- `paramedical courses after 12th`
- `bsc mlt admission`, `cvt course admission`, `ott course college`, `mrit course`

4. `Search | Generic + Geo | State/City`
- `allied health colleges in Bangalore`
- `paramedical colleges in Karnataka`

5. `Search | Competitor / Alternative` (careful with policy/quality)
- Only if landing page has comparison-ready content and legal review is done

6. `DSA | University Pages` (long-tail capture)
- Feed only approved EDGE university landing pages

7. `Remarketing Search (RLSA)` (optional layer)
- Visitors / lead form opens / brochure downloaders

## 7) Meta Ads Strategy (EDGE)

### 7.1 Meta Funnel Design
Run by objective, not one mixed campaign.

1. `TOF | Awareness / Traffic / Video Views`
- Campus + career aspiration + parent reassurance
- Goal: audience warming + engagement pools

2. `MOF | Leads (Instant Form / Website)`
- Program-specific and university-specific messaging
- Goal: qualified lead generation

3. `BOF | Remarketing (Lead to Appointment / Demo Push)`
- Form opens not submitted
- Website visitors
- Video viewers (50%+)
- Existing leads without appointment / no-show / demo pending

4. `Reactivation`
- Old leads (15/30/60/90+ days) by campus status and seat urgency

### 7.2 Meta Campaign Structure (Recommended)
Keep account clean; avoid too many fragmented campaigns.

- `Campaign` = funnel stage + region cluster + objective
- `Ad Set` = geo + audience type + language
- `Ads` = creative angle + program + university

Example naming:
- `IN | META | LEADS | EDGE | SOUTH | TOF_BROAD`
- `IN | META | LEADS | EDGE | KA_BLR | MOF_AHP`
- `IN | META | SALES | EDGE | RMK | FORM_OPENERS_30D`

### 7.3 Meta Audiences
Use simple, scalable architecture.

Prospecting:
- `Broad` (18-24 students)
- `Parents` (35-55) by state cluster
- Interest stacks:
  - healthcare careers
  - nursing / paramedical / physiotherapy
  - colleges / admissions / entrance prep
  - education loans / scholarships

Lookalikes (once data builds):
- 1-3% `Enrolled`
- 1-3% `Demos`
- 1-3% `Appointments`
- 1-3% `High-quality leads`

Remarketing pools:
- Website visitors `7/14/30`
- Lead form opens `7/14`
- Video viewers `50% / 75%`
- CRM custom audiences by stage (`lead`, `appt pending`, `demo pending`)

### 7.4 Meta Creative Angles (What Works for EDGE)
Rotate by audience stage.

Student-first:
- `Career path + salary trajectory + hospital role`
- `Campus life + hostel + labs`
- `Program options after 12th PCB/PCMB`

Parent-first:
- `Accreditation + safety + hostel + placement support`
- `Fee structure + payment cycle + loan support`
- `Clinical training + employability`

Urgency:
- `Admissions open`
- `Limited seats per program`
- `Counselling slots open`
- `Batch starts in August`

## 9) Lead Form Design (Meta + Landing Page)
Keep forms short but qualification-focused.

### Must-have fields
- Student name
- Student phone (WhatsApp)
- Parent phone (optional but recommended)
- City / state
- Class 12 status (`appearing/passed/drop`)
- Stream (`PCB/PCMB/PCM/Other`)
- Program interest
- Preferred university / city
- Budget range (optional in form, strong in call)
- Consent checkbox

### Qualification questions (improves lead quality)
- `Are you open to relocating / hostel?`
- `When do you want to start? (2026 intake)`
- `Do you want a counselling call / campus visit / brochure?`

## 10) Counsellor / Calling Scripts (Execution Scripts)

### 10.1 First Call Script (Lead -> Appointment)
Opening:
- `Hi [Name], this is [Agent] from Emversity admissions support for [University/Program]. You had enquired about allied health admissions for 2026. Is this a good time for 2 minutes?`

Qualification:
- `Which class are you in / completed?`
- `What stream in 12th? PCB/PCMB/PCM?`
- `Which program are you interested in: A&OTT, CVT, MLT, MRIT, BPT, or are you exploring?`
- `Which city/state are you looking for?`
- `Are you open to hostel if the campus is outside your city?`

Pitch (based on fit):
- `Based on your profile, [University Name] is a good option because [hospital exposure / hostel / city / fee structure / intake].`

Appointment close:
- `Let’s schedule a counselling session to explain fees, eligibility, and the admission process in detail.`
- `Would [time slot 1] or [time slot 2] work for you?`

Confirmation:
- `I’ll send a WhatsApp confirmation with program details and documents needed.`

### 10.2 Appointment Reminder (WhatsApp)
- `Hi [Name], reminder for your Emversity counselling session today at [time] for [University/Program]. Please keep your 12th stream/marks details ready. Reply YES to confirm.`

### 10.3 Demo / Counselling Script (Appointment -> Demo)
Agenda:
- Career outcome
- University fit
- Program curriculum + duration
- Eligibility (including NEET if applicable)
- Fees + payment cycle
- Hostel + transport
- Scholarship / loan options
- Admission timeline + seat blocking

Close:
- `Would you like to proceed with application now, or book a campus visit first?`

### 10.4 Enrollment / Payment Script (Demo -> Enrollment)
- `Seats in [Program] at [University] are limited and allocated based on eligibility and document completion.`
- `Today I can help you complete the application and guide the seat-blocking amount/payment process.`
- `Do you want me to send the official payment/application link right now?`

### 10.5 Objection Handling Prompts
- `Fees high`: explain payment cycle, hostel choices, loan support, scholarship eligibility
- `Distance`: highlight hostel safety, transport, campus support
- `Parents unsure`: offer parent counselling call + campus visit
- `NEET confusion`: clarify which programs require NEET and which do not (campus-specific)
- `Job concern`: discuss role pathways, internship, placement support (avoid guaranteed salary claims)

## 14) Governance (To Avoid Chaos Across Campuses)

### Single source of truth per campus
Mandatory approved fields before scaling:
- Admission status
- Start date / exam dates
- Program list and eligibility
- Fee and payment process
- Booking amount + refund status
- Hostel details
- Official application/payment links

### Claims compliance
- No unverified placement guarantees
- No unsupported salary claims
- No misleading scholarship/refund claims
- NEET requirement must be explicit for relevant programs/campuses
