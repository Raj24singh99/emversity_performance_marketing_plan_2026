# Emversity PM 2026 Dashboard

## Run locally

From the project root:

```bash
cd "/Users/rajsingh/Desktop/Performance Marketing Plan 2026/emversity_performance_marketing_2026"
python3 -m http.server 8080
```

Open:

`http://localhost:8080/dashboard/`

## What it includes

- Overview tab: Annual target sheet KPIs and program-level PM budget/enrollment visuals
- Monthly Funnel tab: Month/program/source filters with spend-leads-appointments-demos-enrollment tracking
- Program View tab: Program-level monthly funnel trend and mix
- Scripts tab: Searchable script library with script analytics
